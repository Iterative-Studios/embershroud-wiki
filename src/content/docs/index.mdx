## Two kinds of pages

- **Reference pages** (Skills, Items, Monsters, Zones, NPCs) are **auto-generated from the live game data**, so their stats track the current build. Don't hand-edit the generated sections.
- **Guides and systems pages** are written by the team and community - strategy, theorycraft, and explanations that come from playing.

> Reference content reflects the current public playtest build. Numbers can change between patches; each page shows the build it was generated from.

Spotted something wrong or want to add a guide? See **[how to contribute](/contribute/)**.
